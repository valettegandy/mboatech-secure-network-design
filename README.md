# mboatech-secure-network-design

👉 Titre du projet

Secure Network Design for MboaTech SARL

👉 Domaine choisi

Reseau

👉 Le probléme traité

Organisation du réseau
    le reseau est constitue de plusieur département
    🟡ADMIN
    🟡DEV
    🟡SUPPORT
    🟡GUEST
    🟡SERVERS
    Separation de département en VLAN
    Routage inter-VLAN
    attribution automatique des IP(DHCP)
    sécurité du reseau(ACL)

👉Outils utilisés

 🟡Cisco Packet Tracer: il s'agit d'un logiciel de simulation de réseau informatique
    il principalement utilisé dans l'apprentisage des réseau.
 🟡 Git:c'est systéme de gestion de versionsdistribué utilisé pour suivre les modification dans des fichiers
     surtout dans les projets de développementlogiciel. 
 🟡 Visual Studio Code (souvent appelé VS Code) est un éditeur de code gratuit . Il est utilisé pour écrire, modifier et exécuter du code dans plusieurs langages de programmation.
